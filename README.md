# Note-Lite

**Live Demo:** [https://note-lite.vercel.app/](https://note-lite.vercel.app/)

Note-Lite is a minimalist, responsive, and single user note taking application designed exclusively for focused text input.

-----

## Features

  * **Blazing Fast:** Built with Next.js, ensuring excellent performance and fast loading times.
  * **Theme Toggling:** Supports automatic system theme detection and offers a manual Light/Dark mode toggle for personalized viewing.
  * **Local Persistence:** All notes are automatically saved to your browser's `localStorage` instantlyâ€”no backend or internet connection required after the initial load.
  * **Real-time Editing:** Seamlessly update and search your notes in a clean, focused editor.
  * **Fully Responsive:** Works flawlessly across desktop and mobile devices with an optimized sidebar and editor view.
  * **Keyboard Shortcut:** Quickly create a new note using **Shift + N**, save with **Shift + S**.

-----

## Tech Stack

  * **Framework:** [Next.js (App Router)](https://nextjs.org/)
  * **Styling:** [Tailwind CSS](https://tailwindcss.com/) 
  * **Icons:** [Lucide React](https://lucide.dev/icons/)
  * **Language:** TypeScript
  * **Data Storage:** Browser `localStorage`
